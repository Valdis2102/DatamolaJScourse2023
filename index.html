Clever Clock

Web application (writed by JQuery) for people who spend a lot of time at the computer.

This application will help you control your working time. For example, you can start a timer to take a break or coffee break. You can also set an alarm and set it to run every day (for example, remind yourself of your lunch break). The main condition for this application to work is that the page must be open in your browser.

The application determines your location and current time by IP and uses Local Storage to save alarm settings.

I'm keep working on the project. I plan to rewrite on VUE .js, connect the database, add user authorization and create ToDo List.
